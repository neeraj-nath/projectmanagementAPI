# Project Management API

A RESTful Web API for managing projects, tasks, and users.  
This API is designed to support core project management features such as creating projects, assigning tasks, tracking progress, and managing users.

---

## Tech Stack

- ASP.NET Core Web API (.NET 8)
- Entity Framework Core
- SQL Server (can be replaced with PostgreSQL)
- RESTful architecture
- Swagger / OpenAPI

---

## Features

- Project creation and management
- Task creation, assignment, and status tracking
- User management
- Role-based access (planned / optional)
- Clean layered architecture
